Reset Seashell

It often helps to hard-reset your Seashell instance if it gets locked up in some way. To do this, click the button below to log in again, and check the box labeled "Check to hard-reset your Seashell instance."

Default Editor Shortcuts

These are shortcuts that will work when the editor window is focused (the cursor will be blinking if the window is focused).
  • Ctrl-f Start new search.
  • Ctrl-g Go to next match.
  • Ctrl-G Go to previous match.
  • Ctrl-z Undo.
  • Ctrl-y Redo.
  • Ctrl-j Go to line.
